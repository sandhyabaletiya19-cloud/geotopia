📏 Triangulation & National Surveys

🎯 Great Trigonometric Survey of India

1802-1871

  • 📐 2,400 km baseline measured
  • 🏔️ Calculated Mt. Everest height: 29,002 ft (actual: 29,032 ft)
  • 📊 Proved Earth is oblate spheroid
  • 🗺️ Mapped entire Indian subcontinent
Triangulation Method:
  1. Measure one baseline accurately
  2. Measure angles from endpoints
  3. Calculate other sides using trigonometry
  4. Use calculated sides as new baselines
  5. Repeat to cover entire area

🎨 THEMATIC MAPS

Visualize specific data or themes over geographic area

📊

Choropleth Maps

Most Common
📖 Definition:

Uses different colors/shades within predefined areas (countries, states, counties) to represent data values

✅ Best Practices:
  • 5-7 classes maximum (human color perception limit)
  • Sequential scheme for single variable (light→dark)
  • Diverging scheme for data with meaningful midpoint
  • Qualitative scheme for categorical data
  • Normalize data (use rates/densities, not raw counts)
📐 Data Classification Methods:
Equal Interval

Range divided into equal steps

Example: 0-20, 20-40, 40-60
Quantile

Equal number of features per class

Each class has 20% of data
Natural Breaks (Jenks)

Minimizes within-class variance

Groups similar values
Standard Deviation

Based on distance from mean

±1σ, ±2σ classes
🎯 Common Applications:
📈 Census data 🗳️ Election results 💰 Income levels 🎓 Education rates 🏥 Health statistics 🌡️ Climate zones
⚠️ Common Pitfalls:
  • ❌ Using for point data (use proportional symbols instead)
  • ❌ Not normalizing (showing population instead of density)
  • ❌ Too many classes (visual confusion)
  • ❌ Poor color choice (red-green for colorblind users)
⭕

Proportional Symbol Maps

📖 Definition:

Symbol size varies in proportion to data value at each location

📏 Scaling Methods:
Mathematical Scaling

Area proportional (correct):

radius = √(value/π)

Circle area matches data value

⚠️ NOT diameter proportional

Would make differences appear larger than reality

Perceptual Scaling

Slightly exaggerate differences for better perception

radius = value^0.57

Flannery adjustment for visual perception

🔷 Symbol Shapes:

Circles

✓ Easy to compare

✓ No directional bias

Squares

✓ Fill space efficiently

✗ Harder to compare

Pictograms

✓ Intuitive

✗ Harder to scale accurately

🎨 Design Guidelines:
  • Transparency: 50-70% to show overlapping symbols
  • Outline: Darker stroke for definition
  • Size range: Don't exceed 20:1 ratio
  • Legend: Show 3-5 representative sizes
  • Overlap: Smaller symbols on top
🎯 Best Used For:
🏙️ City populations 🌋 Earthquake magnitudes 💰 Sales by location 🏭 Factory outputs 🌊 Rainfall amounts
••••

Dot Density Maps

📖 How It Works:

Each dot represents a fixed quantity. Dots randomly placed within areas.

Example: 1 dot = 100 people

✅ Advantages:
  • Shows distribution AND magnitude
  • Visually intuitive (more dots = more quantity)
  • Can see concentration patterns
  • Good for large datasets
⚠️ Challenges:
  • Time-consuming to create manually
  • Choosing dot value (too high = too sparse, too low = overcrowding)
  • Difficult to get exact counts from map
  • Dots may overlap in dense areas
🎯 Choosing Dot Value:

Formula:

Dot Value = Total Quantity / (Map Area × Desired Density)

Rule of Thumb: Densest area should be ~50-75% covered with dots

📊 Common Uses:
👥 Population distribution 🌾 Crop production 🐄 Livestock density 🏭 Industrial output
〰️

Isoline (Isopleth) Maps

📖 Definition:

Lines connecting points of equal value

📏 Common Isoline Types:
Isotherm

Equal temperature

🌡️ Climate maps

Isobar

Equal atmospheric pressure

⛈️ Weather maps

Isohyet

🌧️ Rainfall maps

Isogloss

🧮 Creating Isolines:
  1. Collect data at specific points
  2. Interpolate values between points
  3. Draw lines connecting equal values
  4. Label lines with values
  5. Optional: Fill between lines with colors
📖 Reading Isolines:
  • Lines close together = rapid change (steep gradient)
  • Lines far apart = gradual change
  • Closed circles = local maxima/minima
  • Labels show actual values

📖 Definition:

Show movement between locations. Line width proportional to volume.

🎯 Key Elements:
  • Nodes: Origin and destination points
  • Lines: Flow paths (straight or curved)
  • Width: Volume/quantity of flow
  • Arrows: Direction of movement
  • Color: Can show type of flow
📊 Common Uses:
🚚 Trade

Import/export volumes

👥 Migration

✈️ Transportation

Passenger/cargo flows

📱 Communication

Data transfer, calls

💰 Financial

Money transfers

⚡ Resources

📜 Famous Example:

Minard's Napoleon Map (1869)

Shows French army's march to/from Moscow

  • Line width = troop numbers

🔥

Heat Maps

📖 Definition:

Use color intensity to show density or concentration of point phenomena

🎨 Characteristics:
  • Continuous surface from point data
  • Kernel density estimation algorithm
  • Hot colors (red/orange) = high concentration
  • Cool colors (blue/green) = low concentration
  • Smooth gradients between areas
⚙️ Key Parameters:
Search Radius:

Distance over which points influence the surface

Larger radius = smoother, more generalized

Cell Size:

Resolution of output surface

Smaller = more detail, larger file size

💡 Popular Uses:
🚨 Crime analysis 📱 Cell phone usage 👁️ Website click tracking 🦠 Disease outbreaks 🚗 Traffic accidents 🛍️ Customer locations
🆚 Heat Map vs. Choropleth:
Heat Map Choropleth
Input Point data Area data
Boundaries No boundaries Discrete areas
Gradient Smooth, continuous Abrupt changes
Best for Point patterns, clusters Regional statistics
🔄

Cartograms

Distorted
📖 Definition:

Map where area size is distorted to be proportional to a data variable (not geographic area)

🎭 Types of Cartograms:
1. Contiguous Cartogram
  • ✓ Regions remain connected
  • ✓ Topology preserved
  • ✓ Shape distorted
  • Algorithm: Gastner-Newman, Dorling
2. Non-Contiguous Cartogram
  • ✓ Regions can separate
  • ✓ Shape maintained better
  • ✗ Gaps appear
  • Advantage: Easier to recognize regions
3. Dorling Cartogram
  • ✓ All regions = circles
  • ✓ Circle size = data value
  • ✓ Positioned near actual location
  • Advantage: Easy area comparison
4. Demers Cartogram
  • ✓ Rectangular regions
  • ✓ Grid-like arrangement
  • Example: US state election maps
🌟 Famous Examples:
🗳️ Electoral College Maps

US states sized by electoral votes, not land area

🌍 World Population Map

Countries sized by population (China & India huge!)

💰 GDP Cartograms

Countries sized by economic output

✅ Advantages:
  • Visually dramatic
  • Corrects geographic size bias
  • Area comparison intuitive
❌ Disadvantages:
  • Regions may be unrecognizable
  • Confusing if not labeled
  • Can't show precise values

🎓 MASTERING MAP READING

📏 UNDERSTANDING SCALE

1. Verbal Scale

"One inch represents one mile"

Easy to understand but doesn't resize with map

2. Ratio/Representative Fraction (RF)

1:24,000

1 unit on map = 24,000 same units on ground

Common USGS topographic map

1 inch on map = 24,000 inches = 2,000 feet = 0.38 miles
1:100,000

Smaller scale (shows more area, less detail)

1 cm on map = 1 km on ground
1:1,000,000

Very small scale (world atlas)

1 cm on map = 10 km on ground
🔍 Understanding "Large" vs "Small" Scale:
1/1,000
LARGE SCALE

= 0.001 (bigger number)

✓ More detail
✓ Smaller area
✓ City maps

→
1/1,000,000
SMALL SCALE

= 0.000001 (smaller number)

✓ Less detail
✓ Larger area
✓ World maps

3. Graphic/Bar Scale

🧮 Scale Calculations Practice:

Problem 1:

Map scale is 1:50,000. Two cities are 7 cm apart on the map.

What is the actual distance?

Step 1: 7 cm on map × 50,000 = 350,000 cm on ground

Step 2: 350,000 cm ÷ 100,000 = 3.5 km

✓ Answer: 3.5 kilometers

Problem 2:

Actual distance between two points is 12 miles.

On a 1:24,000 map, how far apart will they be?

Step 1: 12 miles × 63,360 inches/mile = 760,320 inches

Step 2: 760,320 ÷ 24,000 = 31.68 inches on map

✓ Answer: 31.68 inches (about 2.6 feet!)

This map would be impractically large. Need smaller scale!

🧭 DIRECTION & ORIENTATION

Cardinal Directions

4 Cardinal: N, E, S, W

4 Intercardinal: NE, SE, SW, NW

8 Secondary: NNE, ENE, ESE, SSE, SSW, WSW, WNW, NNW

Total: 16-point compass

Azimuth/Bearing (Degrees)

Azimuth: Angle measured clockwise from North

  • North = 0° (or 360°)
  • East = 90°
  • South = 180°
  • West = 270°

Used in surveying, navigation, military

Mils (Military)

Definition: 1 mil = 1/6400 of a circle

Why 6400? Makes calculations easier:

At 1000 meters distance:

1 mil = ~1 meter width

Width (m) = Distance (km) × Mils

Conversions:

  • 360° = 6400 mils
  • 1° = 17.78 mils
  • 90° = 1600 mils

Geographic Coordinates (Lat/Long)

Formats for 40.7128°N, 74.0060°W (New York):

UTM (Universal Transverse Mercator)

Structure: Zone + Easting + Northing

Example: 18T 583960mE 4507523mN

18T

Zone (60 zones worldwide, 6° wide)

583960mE

Easting (meters east of zone center)

4507523mN

Northing (meters north of equator)

✅ Advantages:
  • Measurements in meters (easy distance calc)
  • Minimal distortion within zone
  • Used by military, GPS units

MGRS (Military Grid Reference System)

Example: 18TWL8396007523

More concise than UTM, used by NATO forces

Precision by digits:
  • 18TWL: 100km grid square
  • 18TWL83 07: 10km (2 digits)
  • 18TWL8396 0752: 100m (4 digits)
  • 18TWL839960 075230: 1m (6 digits)
  • 18TWL83996025 07523048: 1mm (8 digits)
