⚡ Quick Reference

Core Concepts at a Glance

Key theories and processes you must know

🌍

Plate Tectonics

→ Earth's crust divided into plates

→ Plates float on semi-fluid mantle

→ Movement causes earthquakes, volcanoes

→ 3 boundaries: Convergent, Divergent, Transform

Learn More →
🗺️

Continental Drift

→ Proposed by Alfred Wegener (1912)

→ All continents were once Pangaea

→ Evidence: Fossils, rock formations, climate

→ Led to Plate Tectonics theory

Learn More →
🪨

Rock Cycle

→ Igneous → Sedimentary → Metamorphic

→ Continuous transformation process

→ Driven by heat, pressure, erosion

→ Takes millions of years

Learn More →
🌡️

Weathering & Erosion

→ Physical: Temperature, frost, roots

→ Chemical: Acid rain, oxidation

→ Biological: Plants, animals

→ Shapes all landforms over time
