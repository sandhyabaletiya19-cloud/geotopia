How to Play

🎯 Objective

Fill in the blank with the correct geography answer before time runs out!

📚 Categories

  • Capitals: Name capital cities
  • Countries: Identify nations
  • Landmarks: Famous places
  • Geography: Rivers, mountains, facts
  • Flags: Country by flag description
  • Mixed: All combined

⚡ Scoring

  • Base points: 100 per correct answer
  • Speed bonus: Up to +50 for quick answers
  • Streak multiplier: x2, x3, x4 for consecutive correct
  • Perfect bonus: Extra points for no hints used

💡 Hints

  • 3 hints per game
  • Reveals continent or first letter
  • Reduces points earned
